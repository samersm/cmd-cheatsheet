
### Bash Server
$ Ctrl-C    //To stop

$ Ctrl-D    //To exit

$ Q   //To quit

### Editor
$ Shift + CMD + D   // To duplicate a single line in c9 editor


### Chrome Commands

$ Shift + CMD + R   // Hard refresh

$ ALT + CMD + J     // To open chrome console for debugging

$ CMD + T       // New Tab

$ CMD + W       // Close Tab

$ CTRL + TAB    // Swich Tabs

$ ALT + CMD + (L, R Arrows)     // Switch to Left or Right Tabs
